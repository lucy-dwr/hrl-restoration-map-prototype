# Beta Testing Process

This directory records structured beta testing processes for the HRL restoration
map prototype. Beta test feedback is collected outside the app through an
external form; the form content and review protocol live here so they can be
versioned with the development work they inform.

## Current Round

**Audience:** HRL Habitat Implementation Team, HRL Science Committee  
**Goal:** Identify map, data, content, and usability changes that would make the
prototype more useful and trustworthy for internal and external audiences seeking
an overview of HRL restoration projects.  
**Collection method:** External form shared by email. No in-app feedback button
or frontend persistence is planned for this round.  
**Estimated reviewer time:** 10 to 15 minutes

## Reviewer Instructions

Please complete the tasks below before filling out the form. The tasks are
intended to test the main ways reviewers are expected to use the prototype:
orientation, geographic browsing, project inspection, filtering, context layers,
and overall trust in the data display.

General impressions are also welcome. The most useful feedback, whether
task-specific or general, describes what you noticed, where it happened, why it
matters, and what change would make the map more useful for your work.

Please complete these beta test tasks:

1. **First impression:** Open the map and take one minute to orient yourself
   without using the form or instructions.
2. **Find a familiar area:** Navigate to a river corridor, watershed, bypass, or
   region you know.
3. **Inspect a project:** Click one project and review the information in the
   detail panel.
4. **Browse and filter projects:** Open the Projects tab, search or filter the
   list, select a project, and use at least one map filter.
5. **Use context layers:** Turn at least one watershed, Delta, bypass, stream, or
   basemap layer on or off.
6. **Answer a practical question:** Use the map to answer one or more of these:
   - Where are projects concentrated?
   - What kinds of restoration are represented in an area you know?
   - Which projects would you want to examine further?
7. **Assess trust:** Consider whether the map makes the project data,
   boundaries, acreage, terminology, and general context clear enough for your
   use.
